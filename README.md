# RoomEscape

bangtal 라이브러리를 사용해 C++로 구현한 방탈출 게임 입니다.<br>
게임의 목적은 각 방의 문을 열고 획득한 세 개의 별로 최종 숨겨진 문을 찾아 탈출하는 것입니다. 
첫번째 방에서는 배고픈 주황이에게 사과를 줘서 별을 획득하고 화분에 숨겨진 열쇠를 찾아 문을 엽니다.

두번째 방에서는 상자를 계속 클릭하면 상자가 조금씩 회전하면서 안에 있던 힌트 카드가 나옵니다. 그 힌트로 문의 비밀번호를 알 수 있습니다.<br>
<div>
<img width="80%" height="80%" src="https://user-images.githubusercontent.com/54932132/92625890-2e464f80-f304-11ea-9d16-b977cc3abcde.PNG">
</div>

불을 끄면 숨겨져 있던 별을 획득할 수 있습니다.
<div>
<img width="80%" height="80%" src="https://user-images.githubusercontent.com/54932132/92625905-31d9d680-f304-11ea-8ad4-beeaf478e6a1.PNG">
</div>

마지막 세번째 방에서도 별을 획득하면 총 세 개의 별이 주머니 안에 있습니다.
모든 별을 삼각형에 위치시키면 바닥에서 블랙홀이 열려 탈출 할 수 있습니다. 
<div>
<img width="80%" height="80%" src="https://user-images.githubusercontent.com/54932132/92625913-343c3080-f304-11ea-81fd-a5f62bae2741.PNG">
</div>

바닥의 블랙홀을 클릭하면 탈출에 성공했다는 안내가 나오며 게임이 끝나게 됩니다.
